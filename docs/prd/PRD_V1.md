# K-Beauty Whisperer — 서비스 기획서 (PRD V1)

> **버전**: V1  
> **작성일**: 2026년 2월 26일  
> **문서 성격**: 서비스 전체 기획 개요 (CEO 검토용)

---

## 1. 서비스 개요

**K-Beauty Whisperer**는 K-뷰티 제품을 찾을 때 복잡한 카테고리나 성분 지식 없이도, 일상적인 말로 내게 맞는 제품을 추천받을 수 있는 AI 기반 뷰티 검색 서비스입니다.

> "하루종일 촉촉한데 끈적이지 않는 선크림 있나요?"  
> "민감 피부인데 레티놀 써도 되는 세럼"  
> "쿨톤에 어울리는 봄 립 추천해줘"

이런 식으로 입력하면 AI가 의도를 이해하고, 조건에 맞는 제품을 바로 추천해 줍니다.

---

## 2. 핵심 가치

| 가치 | 설명 |
|------|------|
| **쉬운 검색** | 카테고리 탭을 뒤지거나 필터를 설정할 필요 없이 말하듯이 검색 |
| **개인화된 추천** | 피부 타입, 톤, 고민 등 내 정보를 저장해 두면 더 잘 맞는 결과 제공 |
| **전문가 수준의 분석** | 단순 검색 결과가 아닌 "왜 이 제품이 나에게 맞는지"까지 설명 |
| **프리미엄 루틴 리포트** | 내 피부에 맞는 아침/저녁 루틴을 AI가 작성해 유료로 제공 |

---

## 3. 주요 이용자

- **20대 여성**, 피부 트러블·성분에 민감해 제품 고르기가 어려운 사람
- **K-뷰티에 관심 있는 해외 소비자**, 한국 뷰티 제품을 감성·무드 기반으로 탐색하고 싶은 사람

## 5. 필요한 화면 (페이지 목록)

### 일반 사용자 화면

| 화면 | 접근 | 설명 |
|------|------|------|
| **홈** | 누구나 | 서비스 진입점. 검색창과 예시 문장·예시 태그 제공. 로그인 시 최근 검색 내역 표시 |
| **검색 결과** | 누구나 | 자연어 검색 후 AI가 분석한 결과와 제품 목록 표시. 카테고리·가격대·성분 등 필터 가능 |
| **제품 상세** | 누구나 | 제품 정보, 추천 이유, 주의 성분, 유사 제품 등 상세 안내. 저장하기 버튼 |
| **저장 목록** | 로그인 필요 | 마음에 들었던 제품을 모아보는 화면. 비교 기능 포함 |
| **내 계정** | 로그인 필요 | 피부 타입·톤·고민 등 내 정보 설정 및 검색 이력 관리 |
| **AI 루틴 리포트** | 로그인 + 결제 | 내 피부에 맞춘 아침/저녁 루틴, 추천 근거, 주의사항을 담은 맞춤 리포트 |
| **로그인 / 회원가입** | 비로그인 | 이메일로 가입 또는 로그인, 비밀번호 재설정 |

### 관리자 화면 (운영팀 전용)

| 화면 | 설명 |
|------|------|
| **대시보드** | 서비스 주요 지표 한눈에 확인 (검색량, 인기 검색어, 카테고리 분포 등) |
| **제품 관리** | 제품 등록·수정·삭제 및 목록 조회 |
| **사용자 관리** | 가입자 목록 조회 및 상태 관리 |
| **검색 로그** | 사용자들이 어떤 검색어를 입력했는지 이력 확인 |
| **서비스 설정** | 홈 예시 문장, AI 추천 개수, 리포트 가격 등 설정 변경 |

## 6. 주요 기능 흐름

### 검색부터 추천까지

```
사용자가 자연어로 입력
    ↓
AI가 의도 분석 (어떤 피부 타입? 어떤 효과? 어떤 성분 제외?)
    ↓
조건에 맞는 제품 목록 표시 + 각 제품에 "왜 나에게 맞는지" 설명
    ↓
제품 상세 페이지에서 더 자세히 확인
    ↓
저장하거나, 프리미엄 루틴 리포트 구매
```

### 프리미엄 루틴 리포트 흐름

```
검색 결과 또는 제품 상세에서 "내 루틴 만들기" 버튼 클릭
    ↓
결제 진행
    ↓
AI가 내 피부 정보 + 검색 맥락을 바탕으로 루틴 리포트 생성
    ↓
아침/저녁 단계별 루틴, 추천 근거, 주의사항 확인
```

## 7. 기술 구성 (개요)

기술적 세부 구현은 별도 문서를 참고하며, 여기서는 핵심 구성만 간략히 안내합니다.

| 영역 | 사용 기술 |
|------|----------|
| 화면(프론트엔드) | React |
| 데이터 저장 및 사용자 인증(백엔드/DB) | Supabase |
| 자연어 문장 검색 | Supabase 벡터 DB (의미 기반 문장 검색) |

> **벡터 DB 문장 검색이란?**  
> "촉촉하지만 가볍고 끈적이지 않는 느낌"처럼 딱 떨어지는 단어가 없어도, 비슷한 의미를 가진 제품 설명을 AI가 찾아주는 방식입니다. 기존 키워드 검색보다 자연스러운 표현으로 검색할 수 있습니다.
